# WannaCrypt0r Project
해당 프로젝트는 오직 연구를 목적으로 제작한 것이고 이를 통해 아무런 금전적 이득을 취하지 않았습니다. 또한 이 프로젝트를 그 어떠한 범죄 행위라도 허용하지 않습니다. 이를 통해 문제가 발생할 경우 책임지지 않습니다.
